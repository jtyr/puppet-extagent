Puppet-extagent

Puppet module for deployment of the SNMP Extension Agent

Download .zip Download .tar.gz View on GitHub

Description

Puppet module for deployment of the SNMP Extension Agent (Extagent). Extagent is a set of monitoring sensors which extend the functionality of the standard SNMP daemon.

Installation

$ git clone https://github.com/jtyr/puppet-extagent.git
